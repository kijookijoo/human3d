## Run locally

Option A: Python HTTP server
```bash
cd ~/Projects/Human3D
python3 -m http.server 5173
# open http://localhost:5173
```

Option B: Node (serve)
```bash
cd ~/Projects/Human3D
npx serve . -l 5173 --single --yes
# open http://localhost:5173
```
